# Image Slider

A responsive and accessible image slider built with HTML, CSS, and Vanilla JavaScript.

## Features
- Smooth sliding animation
- Previous & Next navigation
- Dynamic bullet navigation
- Keyboard support (Arrow keys)
- Slide counter
- Clean and organized code structure

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
